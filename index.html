Welcome to OneOcean UUV Simulation Demo

This is an interactive underwater simulation demo for ocean robotics scene prototyping.

What you can see: continuous seabed terrain, deep trenches, dense seagrass/coral areas, pollutant targets, and dynamic current particles.

  • Vehicle: UUV/AUV body with rear ducted propellers and spotlights
  • Environment: larger map with varied relief and localized terrain features
  • Water: animated surface waves + depth-dependent lighting attenuation
  • Tasks: measure distance to floating and underwater pollutant targets

Controls remain aligned with the reference online demo.

Quick Start:
W/S: Pitch (forward/backward)
A/D: Roll (left/right)
↑/↓: Altitude
←/→: Yaw rotation
Controls use a single Manual mode (keyboard).
